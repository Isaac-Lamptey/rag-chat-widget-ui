
# RAG Chat Widget UI

A lightweight chat widget UI built for Retrieval-Augmented Generation (RAG) chatbots and AI assistants.

## Features

- Clean floating chat widget
- Responsive design
- Chat interface with user and bot messages
- Easy integration with backend APIs
- Suitable for RAG and customer support assistants
